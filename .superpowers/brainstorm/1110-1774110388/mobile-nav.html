Mobile Navigation Pattern

The app currently has a fixed sidebar — which navigation style do you want on mobile?

A — Bottom Tab Bar

Native mobile feel. 5 primary tabs always visible. Most common on iOS/Android apps. Extra pages tucked in a "More" tab.

B — Slide-out Drawer

Hamburger icon opens a full nav drawer from the left. Full sidebar visible on desktop/tablet. Clean, no persistent nav chrome.

C — Adaptive (Recommended)

Bottom tabs on mobile, collapsible sidebar on tablet, full sidebar on desktop. One codebase, context-aware navigation.

Note: The app currently uses a fixed desktop sidebar only — no mobile support exists yet. Option C gives the best cross-platform experience.
